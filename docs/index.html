ПЛЕЕР DIGITAL SIGNAGE

вещай прямо из Dropbox, GoogleDrive, OneDrive и других популярных облачных хранилищ

Предельно простое решение задач группового вещания  с синхронизацией контента прямо из облака. Для вещания достаточно разместить папку с медиафайлами в Google Drive, DropBox, OneDrive, Яндекс Диск или в других популярных облачных хранилищах. Медиафайлы, размешенные в облачном хранилище, в папке плеера, автоматически синхронизируются по сети и проигрываются на плеере в цикле 24/7. Интернет требуется только для синхронизации нового контента.

Digital Signage, который не нужно изучать

Используй бесплатно - ни каких подписок и скрытых платежей

Свобода и независимость в подарок - работает без cms и сервера

  • Поддержка всех популярных форматов видео, изображений  и WEB контента
  • Автоматическая синхронизация по сети
  • Вещание в группе

90% задач рекламного вещания на экранах и видеостенах формата Digital Signage сводятся к показу видеороликов, и изображений в цикле.

С сетевым рекламным плеером OneBlending Digital Signage решать эти задачи очень просто и удобно.

Оборудование

  • Функционал сетевого плеера OneBlending

- Простое управление контентом через папку в Google Drive, OneDrive, Dropbox, Яндекс Диск;

- Автоматическая синхронизация плеера с папкой облака;

- Подключение неограниченного количества плееров к папке облака - групповое вещание;

- Воспроизведение в цикле, с сортировкой файлов по имени;

- Автоматическое вещание при включении питания плеера, в режиме 24/7;

- Простой функционал назначение хронометража демонстрации изображений, web и количества повторов. Автоматическая генерация отчетности прямо в облако.

- Специально сконфигурированное ПО OneBlending и ОС Windows® 10 Enterprise LBTS , с UWF фильтрацией обеспечивает режим работы - включил и забыл.

  • Медиа-файлы и любое WEB содержание

- Поддержка всех популярных видео-форматов и изображений - MPEG-1/2, DivX® (1/2/3/4/5/6), MPEG-4 ASP, XviD, 3ivX D4, H.261, H.263 / H.263i, H.264, H265 / MPEG-4 AVC, MJPEG (A/B), WMV 1/2, WMV 3 / WMV-9 / VC-1, BMP, JPG, PNG, GIF. Разрешение до 3840×2160@60Hz. Аудио не поддерживается.

 

- Поддержка web содержимого - URL ссылки на web станицы, готовые проекты HTML в виде собранного zip проекта.
